## Quick Reference Commands

### Most Common Commands
```bash
# Development
npm run dev                              # Start all services
npm run dev --workspace=packages/backend # Backend only
npm run dev --workspace=apps/web        # Web only
cd apps/mobile && npm start             # Mobile (Expo)

# Database
npm run migrate                          # Run migrations
npm run seed                             # Seed data
npm run db:reset                         # Reset database

# Testing
npm test                                 # Run all tests
npm run test:integration                # Integration tests
npm run test:e2e                        # End-to-end tests

# Building
npm run build                            # Build all
docker build -f docker/Dockerfile.backend -t puraestate-backend:latest .

# Deployment
npm run deploy:staging                   # Deploy to staging
npm run deploy:prod                      # Deploy to production
```

### Verification Commands
```bash
# Check services
curl http://localhost:3000/api/health   # Backend
curl http://localhost:3001              # Web app
npm run db:verify                        # Database
npm run redis:health                     # Redis
```
