KEY INSIGHTS & LEARNINGS EMBEDDED IN THE PLAN
===============================================================================

1. MARKET INSIGHT
   └─ Costa Rican luxury real estate market is $3B+ annual
   └─ <5% of transactions digital (huge opportunity)
   └─ Users prefer WhatsApp over email (critical feature)
   └─ Mobile adoption 10x web visits (mobile-first required)

2. PRODUCT INSIGHT
   └─ Web failed because of UX friction (copy phone numbers, no notifications)
   └─ Mobile wins because it's native (push, camera, location, biometric)
   └─ Guidlyer's model (AI + subscriptions) works here perfectly
   └─ Agents want subscriptions, not per-transaction fees

3. BUSINESS INSIGHT
   └─ Premium positioning (focus on $400K+ properties)
   └─ Agent subscriptions are scalable revenue (not per-listing)
   └─ Costa Rica focus means deep expertise = competitive moat
   └─ Redemption narrative = great marketing angle

4. EXECUTION INSIGHT
   └─ 30 days is achievable but tight (no feature creep allowed)
   └─ MVP scope is carefully defined (what's in, what's out)
   └─ Risk mitigation has fallbacks (SINPE + Stripe, Claude + GPT)
   └─ Post-launch plan ensures continuous improvement

5. TEAM INSIGHT
   └─ 10-person team is lean but complete
   └─ Roles are clearly defined (no ambiguity)
   └─ Daily standups keep alignment
   └─ Contractors allow flexibility post-launch

===============================================================================
CRITICAL DECISIONS EMBEDDED IN THE PLAN
===============================================================================

WHY REACT NATIVE (not native iOS/Android):
→ 95% code sharing between platforms
→ Same team maintains both
→ JavaScript developers abundant
→ Faster time to market (30 days possible)

WHY MOBILE (not PWA or web):
→ Push notifications (critical for hot deals)
→ Biometric auth (Face ID, Touch ID)
→ Camera integration (property photos)
→ Location services (nearby properties)
→ Home screen icon (higher engagement)

WHY GUIDLYER MODEL:
→ AI matching (OpenRouter Claude)
→ Automations (Composio WhatsApp)
→ Subscription revenue (agent tiers)
→ Costa Rica focus (existing expertise)

WHY SUBSCRIPTION (not per-transaction):
→ Predictable recurring revenue
→ Aligns incentives (agents want quality)
→ Higher lifetime value
→ Easier to justify premium support

WHY SINPE MÓVIL (+ Stripe backup):
→ Local payment method (frictionless for Costa Ricans)
→ Only platform with native integration
→ Competitive advantage
→ Stripe for international fallback

WHY OPENROUTER (for AI):
→ Multi-model support (Claude 3 + GPT-4)
→ Cost optimization (route to cheapest model)
→ Fallback capability (critical in MVP)
→ No vendor lock-in

WHY 30 DAYS:
→ Market opportunity won't wait 6 months
→ Team commitment easier with short sprint
→ MVP scope prevents scope creep
→ Faster feedback loop from real users
→ Competitive advantage (be first to market)
